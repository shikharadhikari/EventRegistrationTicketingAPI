Event Registration & Ticketing API

Create an API to manage events, attendees, and ticket bookings for an event management platform.

Requirements
	•	Register/login users; roles: Organizer, Attendee
	•	CRUD operations on events (Organizer only)
	•	Users can:
	•	View events
	•	Search, filter (category, date, venue), paginate
